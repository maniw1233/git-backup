== Frequently Asked Questions ==

= How to install wp-bizone theme=
1. Login your Admin WordPress dashboard.
2. Go to "Appearance" -> "Theme" and click on "Add New" button.
3. Click on "Upload Theme" button and choose the theme's zip file you have downloaded.
4. Click on "Install Now" button.
5. Activate the theme.
6. Now go to "Appearance" -> "Customize" to configure theme options.

Alternatively,

1. Login to your WordPress dashboard.
2. Go to "Appearance" -> "Theme" and click on "Add New" button.
3. Search for "wp-bizone".
4. Click on Install and Activate.

== Upgrade Notice ==
Update will replace you theme files. For not lost nothing important - Please before update do full website backup(files and database).

= How to Upgrade wp-bizone theme=

1. Backup your files and database - with that ypu not lost nothing important.
2. Login your Admin WordPress dashboard.
3. Go to "Appearance" -> "Theme" and click on "Add New" button.
4. Click on "Upload Theme" button and choose the theme's zip file you have downloaded.
5. Click on "Install Now" button.
6. Press "Replace active with uploaded"
7. Now go to "Appearance" -> "Customize" to configure theme options.

Alternatively,

1. Backup your files and database - with that ypu not lost nothing important.
2. Login to your WordPress dashboard.
3. Go to "Appearance" -> "Theme" and click on "Add New" button.
4. Search for "wp-bizone".
5. Click on Install and Activate.
